
==================
Mini Project
==================

Requirement : User management with below functionalities

1) User Registration (with dependent drop downs)

2) Email Sending upon successfull registration with temp pwd

3) Login

4) Reset Password

5) Dashboard (Random Quote display)

##### Third Party API To get Random Quote : https://dummyjson.com/quotes/random


================
Database tables
================

1) COUNTRY_MASTER

		-- country_id 		INT			PK		AUTO_INCREMENT
		-- country_name		VARCHAR

2) STATE_MASTER		

		-- STATE_ID			INT 		PK 		AUTO_INCREMENT
		-- STATE_NAME 		VARCHAR		
		-- COUNTRY_ID 		INT 		FK		(COUNTRY_MASTER)

3) CITY_MASTER

		-- CITY_ID 			INT 		PK 		AUTO_INCREMENT
		-- CITY_NAME 		VARCHAR
		-- STATE_ID			INT 		FK 		(STATE_MASTER)		

4) USERS_TBL

		-- USER_ID 			INT 		PK 		AUTO_INCREMENT
		-- UNAME 			VARCHAR
		-- EMAIL			VARCHAR
		-- PWD 				VARCHAR		
		-- PWD_UPDATED		VARCHAR
		-- COUNTRY_ID		INT 		FK		COUNTRY_MASTER
		-- STATE_ID			INT 		FK 		STATE_MASTER
		-- CITY_ID			INT 		FK 		CITY_MASTER
		-- CREATE_DATE		DATE
		-- UPDATED_DATE 	DATE

======================
Development Process
======================

1) Create spring-boot app with below dependencies

		a) web-starter
		b) data-jpa
		c) mysql-driver
		d) mail-starter
		e) devtools
		f) project lombok
		g) model mapper
		h) springdoc-openapi-starter-webmvc-ui

2) Create Entities and Repositories with proper association mapping

3) Create DTO classes 

4) We need to create Mapper classes

5) Create Service Interface & impl class with required methods

6) Create RestController with http methods to handle request & respons

7) Configure data source & SMTP props in application.yml file

8) Run the application and test it 

Note: We need to create Postman collection to test our backend api.
